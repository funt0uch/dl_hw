# 🧠 Multi-Modal Cardiac Lesion Segmentation on MyoPS 2020

## 📌 Project Overview

This project focuses on **cardiac lesion segmentation** using multi-modal MRI data from the MyoPS 2020 dataset.
The goal is to accurately segment myocardial pathologies (e.g., scar, edema) by leveraging complementary information across different imaging modalities.

We propose a **multi-modal fusion framework with attention mechanisms** to improve segmentation performance over standard baselines.

---

## 🗂️ Dataset

* Dataset: MyoPS 2020 (Myocardial Pathology Segmentation Challenge)
* Modalities:

  * bSSFP
  * T2
  * LGE
* Task:

  * Multi-class segmentation of myocardial lesions

---

## 🏗️ Methodology

### 1. Baseline Model

We adopt a standard **U-Net architecture** as the baseline for medical image segmentation.

---

### 2. Proposed Method

Our approach introduces two key improvements:

#### 🔹 Multi-Modal Fusion

We explore different fusion strategies:

* **Early Fusion**: Concatenate modalities at input level
* **Late Fusion**: Encode each modality separately before merging
* **Attention Fusion (Proposed)**: Learn adaptive weights for each modality

---

#### 🔹 Attention Mechanism

We integrate attention modules to enhance feature representation:

* Channel Attention
* Spatial Attention

This allows the network to focus on more informative regions and modalities.

---

### 3. Loss Function

To address class imbalance in medical segmentation:

* Dice Loss
* Cross Entropy Loss
* (Optional) Focal Loss

Final loss:
[
\mathcal{L} = \lambda_1 \cdot \mathcal{L}*{Dice} + \lambda_2 \cdot \mathcal{L}*{CE}
]
